= Office-stamper CLI

The `cli` module provides a command-line interface to the Office-stamper engine.
It allows for stamping DOCX (Word) and PPTX (PowerPoint) templates from various input data formats.

== Key Features

* **Multiple Data Formats**: Supports JSON, CSV, Java properties, XML/HTML, and XLSX inputs.
* **Format Support**: Choose between Word (.docx) or PowerPoint (.pptx) templates.
* **Diagnostics**: Built-in diagnostic mode for environment reporting.
* **Native Packaging**: Support for building Windows EXE installers and portable binaries.

== Usage

[source,bash]
----
officestamper \
  -d data.json \
  -t template.docx \
  -o output.docx
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.office-stamper</groupId>
    <artifactId>cli</artifactId>
    <version>${project.version}</version>
</dependency>
----

== Windows Packaging

You can build a native Windows EXE installer or a portable single-file EXE using specific Maven profiles.

=== EXE Installer
[source,powershell]
----
mvn -pl :cli -am -P windows-exe -DskipTests package
----

=== Portable EXE
[source,powershell]
----
mvn -pl :cli -am -P windows-portable -DskipTests package
----
