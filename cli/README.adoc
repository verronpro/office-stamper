= Office-stamper CLI Module

The `cli` module provides a command line interface to the Officeâ€‘stamper engine.
It stamps DOCX (Word) and PPTX (PowerPoint) templates from a variety of input formats.

== Features

* Stamp templates from JSON, CSV, Java properties, XML/HTML and XLSX (Excel) inputs
* Choose the target format: Word or PowerPoint
* Diagnostic context and template bundled for quick environment reports

== Usage

[source,bash]
----
officestamper \
  --input data.json \
  --template template.docx \
  --output output.docx \
  --stamper word
----

Options:

* `-i, --input` (required): input data file or `diagnostic`
* `-t, --template` (required): template file or `diagnostic`
* `-o, --output` (default: `output.docx`): output path
* `-s, --stamper` (default: `word`): `word` or `powerpoint`

== Module name

`pro.verron.officestamper.cli`
