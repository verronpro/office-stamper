= Office-stamper Excel Context

The `excel-context` module allows using Excel (.xlsx) files as data sources for Office-stamper.
It parses Excel sheets and rows to provide a context object that can be used by the engine for stamping templates.

== Key Features

* **Excel Integration**: Use familiar spreadsheet software to manage your template data.
* **Flexible Mapping**: Maps sheets and rows to data structures compatible with SpEL.
* **Simple API**: Easy to load data and pass it to the `engine`.

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.office-stamper</groupId>
    <artifactId>excel-context</artifactId>
    <version>${project.version}</version>
</dependency>
----
