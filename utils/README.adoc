= Office-stamper Utils

The `utils` module contains low-level utilities shared by the Office-stamper engine and tools.
It provides helpers for WordprocessingML (Docx4J) manipulation, iterator utilities, and functional helpers.

== Key Features

* **WML Helpers**: Factories and utilities for working with WordprocessingML and Docx4J.
* **Iterator Utilities**: Specialized iterators like `SlicingIterator` for robust document segmenting.
* **Functional Helpers**: Common functional interfaces and utility methods.
* **Bit/Byte Helpers**: Low-level utilities for data manipulation.

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.office-stamper</groupId>
    <artifactId>utils</artifactId>
    <version>${project.version}</version>
</dependency>
----
